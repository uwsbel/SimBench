```python
import pychrono as chrono
import pychrono.irrlicht as vis
import numpy as np

# Initialize Chrono environment
chrono.SetChronoDataPath("path/to/chrono/data")  # Replace with your Chrono data path
system = chrono.ChSystemNSC()

# Create rigid terrain
terrain = chrono.ChBodyEasyBox(10, 10, 0.1, 1000)  # Adjust dimensions as needed
terrain.SetPos(chrono.ChVectorD(0, 0, 0))
terrain.SetBodyFixed(True)
system.Add(terrain)

# Load custom texture for terrain
terrain_texture = vis.IrrlichtTexture("path/to/terrain/texture.png")  # Replace with your texture path
terrain.GetVisualShape().SetTexture(terrain_texture)

# Create CityBus vehicle
bus = chrono.ChBodyEasyCylinder(0.5, 2, 1000)  # Adjust dimensions as needed
bus.SetPos(chrono.ChVectorD(0, 0, 0.5))
bus.SetRot(chrono.Q_from_Euler(0, 0, 0))
system.Add(bus)

# Add wheels to the bus
for i in range(4):
    wheel = chrono.ChBodyEasyCylinder(0.1, 0.2, 100)  # Adjust dimensions as needed
    wheel.SetPos(bus.GetPos() + chrono.ChVectorD(0, 0, 0.1) + chrono.ChVectorD(0.5 * i - 1, 0, 0))
    wheel.SetRot(chrono.Q_from_Euler(0, 0, 0))
    wheel.SetBodyFixed(True)
    system.Add(wheel)

# Set tire model
tire_model = chrono.ChTireModel(
    bus,
    wheel,
    0.1,  # Tire radius
    1000,  # Tire stiffness
    0.5,  # Tire damping
)
system.Add(tire_model)

# Create Irrlicht visualization
vis_app = vis.ChIrrApp(system, "CityBus Simulation")
vis_app.AddCamera(
    chrono.ChVectorD(5, 5, 2), chrono.ChVectorD(0, 0, 0), chrono.ChVectorD(0, 1, 0)
)
vis_app.SetCamera(0)

# Interactive driver system
steering_angle = 0
throttle = 0
brake = 0

def on_key_pressed(key):
    global steering_angle, throttle, brake
    if key == vis.KEY_LEFT:
        steering_angle -= 0.1
    elif key == vis.KEY_RIGHT:
        steering_angle += 0.1
    elif key == vis.KEY_UP:
        throttle = 0.5
    elif key == vis.KEY_DOWN:
        brake = 0.5

def on_key_released(key):
    global steering_angle, throttle, brake
    if key == vis.KEY_UP:
        throttle = 0
    elif key == vis.KEY_DOWN:
        brake = 0

vis_app.SetKeyCallback(on_key_pressed, on_key_released)

# Simulation loop
while vis_app.Run():
    system.DoStepDynamics(1.0 / 50.0)  # Update at 50 FPS
    vis_app.Render()

    # Apply steering, throttle, and brake to the vehicle
    bus.SetWheel(0, steering_angle)
    bus.SetWheel(1, steering_angle)
    bus.SetWheel(2, steering_angle)
    bus.SetWheel(3, steering_angle)
    bus.SetMotorSpeed(0, throttle)
    bus.SetMotorSpeed(1, throttle)
    bus.SetMotorSpeed(2, throttle)
    bus.SetMotorSpeed(3, throttle)
    bus.SetBrake(0, brake)
    bus.SetBrake(1, brake)
    bus.SetBrake(2, brake)
    bus.SetBrake(3, brake)



```

**Explanation:**

1. **Initialization:**
   - Imports necessary libraries.
   - Initializes the Chrono environment and sets the data path.
   - Creates a `ChSystemNSC` object to manage the simulation.

2. **Rigid Terrain:**
   - Creates a `ChBodyEasyBox` object representing the terrain.
   - Sets its position, dimensions, and mass.
   - Makes it fixed by setting `SetBodyFixed(True)`.
   - Adds the terrain to the simulation system.
   - Loads a custom texture for the terrain using `IrrlichtTexture` and applies it to the terrain's visual shape.

3. **CityBus Vehicle:**
   - Creates a `ChBodyEasyCylinder` object for the bus body.
   - Sets its position, orientation, and mass.
   - Adds the bus to the simulation system.
   - Creates four `ChBodyEasyCylinder` objects for the wheels.
   - Positions the wheels relative to the bus body.
   - Sets the wheels as fixed to the bus.
   - Adds the wheels to the simulation system.

4. **Tire Model:**
   - Creates a `ChTireModel` object to simulate the tire behavior.
   - Specifies the bus body, wheel, tire radius, stiffness, and damping.
   - Adds the tire model to the simulation system.

5. **Irrlicht Visualization:**
   - Creates an `ChIrrApp` object to handle visualization using Irrlicht.
   - Adds a camera to the scene and sets its initial position, target, and up vector.
   - Sets the camera to be the active one.

6. **Interactive Driver System:**
   - Defines functions `on_key_pressed` and `on_key_released` to handle keyboard input.
   - Updates `steering_angle`, `throttle`, and `brake` variables based on key presses and releases.
   - Sets the key callback for the Irrlicht application to handle input events.

7. **Simulation Loop:**
   - Enters a loop that runs until the Irrlicht application is closed.
   - Calls `system.DoStepDynamics()` to update the simulation for one time step (1/50 seconds).
   - Calls `vis_app.Render()` to render the scene.
   - Applies the steering, throttle, and brake values to the bus's wheels and motors.



**Remember:**

- Replace `"path/to/chrono/data"` with the actual path to your Chrono data directory.
- Replace `"path/to/terrain/texture.png"` with the path to your custom terrain texture.
- Adjust the vehicle dimensions, tire parameters, and other settings as needed for your specific simulation.
